# RDR3 AI Vibe Project

I am vibe-coding a Red Dead Redemption 3-style open world in Godot 4.4.
Use only GDScript.
Project structure:
- player/ - character, horse, controllers
- world/ - terrain, buildings
- systems/ - inventory, dialogue, etc.

Style: realistic western, beautiful camera, smooth movement.
Always create .tscn scenes together with scripts.
